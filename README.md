# 🦉 YTV MAGAZA OwO KAYIT BOTU

![Version](https://img.shields.io/badge/version-1.0-blue)
![Python](https://img.shields.io/badge/Python-3.8+-green)
![Security](https://img.shields.io/badge/Security-KeyAuth_Protected-red)

**YTVMAĞAZAofficial** tarafından geliştirilen, OwO botu için gelişmiş kayıt botu sistemidir.

## ✨ Özellikler
- **🔒 HWID Lock:** Lisanslar donanıma kilitlenir, paylaşılamaz.
- **⏳ Otomatik Süre Yönetimi:** 30 günlük veya sınırsız lisans seçenekleri.
- **🤖 İnsan Benzeri Davranış:** Özelleştirilebilir `min_wait` ve `max_wait` süreleri.
- **📂 Kolay Yapılandırma:** `config.json` üzerinden tek tıkla ayar imkanı.
- **🖱 Buton Algılama:** OwO'nun butonlarını otomatik onaylar.

## 🚀 Kurulum

1. Depoyu indirin veya satın aldığınız dosyaları klasöre çıkartın.
2. Eğer bilgisayarınızda Python[Git](https://www.python.org/downloads/) yüklü değilse indirin
2. `kurulum.bat` dosyasını çalıştırarak bağımlılıkları yükleyin.
3. `config.json` dosyasını düzenleyin:
   ```json
   {
       "LISANS_ANAHTARI": "LİSANS_ANAHTARINIZ",
       "KANAL_ID": 123456789,
       "MIN_BEKLEME": 10.0,
       "MAX_BEKLEME": 17.0
   }
4. `tokens.txt` içerisine tokenleri ekleyiniz.
5. Programı çalıştırın.
